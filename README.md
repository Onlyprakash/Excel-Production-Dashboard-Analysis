# Excel Production Dashboard Analysis

## Project Overview
This project focuses on analyzing production performance and operational trends using Microsoft Excel dashboards, Pivot Tables, and interactive visualizations.

The dashboard helps monitor:
- Production cost trends
- Product-wise analysis
- Manager task distribution
- Production volume over time
- Average production cost insights

# Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
- Data Analysis

# Dashboard Features
- Interactive slicers for filtering
- Production trend analysis
- Product category comparison
- Task distribution analysis
- Average production cost visualization

# Key Insights
- Automobiles showed the highest production cost.
- Production volume peaked during specific months.
- Task distribution varied significantly across managers.
- Furniture category showed high average production cost per unit.
